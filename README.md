> A client powered by Vue 3. It uses Bootstrap 5 as the UI library and icons powered by Bootstrap Icons.
> The server is built using NodeJS and Express. Both client and server portions of the app support 
> Typescript as well as regular Javascript.

#### Serve client and server for development and build for production
 - `npm run serve` to run both client & server project concurrently
 - `npm run build` to build both client & server project to dist/

#### Development build for both client and server
 - `npm run dev:client` to run client project.
 - `npm run dev:server` to run server project.  


#### Production build or both client & server
 - `npm run build:client` to build client project.
 - `npm run build:server` to build server project.
